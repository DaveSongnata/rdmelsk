Instalação
Clone o repositório:
```bash
git clone https://github.com/seu-usuario/guide.git
```
Instale as dependências:
```bash
cd guide
npm install
```
Configure o Firebase:
Crie um arquivo firebase-config.json em src/services/
Adicione suas credenciais do Firebase
Inicie o projeto:
```bash
npm start
```
